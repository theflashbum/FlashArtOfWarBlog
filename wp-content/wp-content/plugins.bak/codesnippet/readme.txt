Code Snippet is WordPress plugin for displaying code with highlighting in blog posts.
Plugin uses GeSHi syntax highlighter engine.

For updates see http://blog.enargi.com

Installation:
- copy archive to wp-content/plugins directory
- extract the zip file
- when extracted properly you should have following directory structure: /wp-content/plugins/codesnippet
Plugin will not work if extracted/copied in diffrent directory.
