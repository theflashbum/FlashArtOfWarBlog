== Installation ==

1. Upload `lightbox.php` and "lightbox folder" to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Add a rel="lightbox" attribute to any link tag to activate the lightbox or rel="lightbox[roadtrip]" for image set
