Apply reflection effect to images with 'reflection' as one of the HTML attribute class values.

== Description ==
Reflection is a WordPress plugin which applies reflection effect to images with 'reflection' as one of the HTML attribute class values.

This plugin uses [Raphael](http://raphaeljs.com/) JavaScript library to create the reflection effect.

For Installation, Usage, and other information, please visit [Reflection Plugin](http://code.google.com/p/bitpress/wiki/Reflection) web page.
